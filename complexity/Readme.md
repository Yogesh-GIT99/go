# Time and Space Complexity

Time Complexity:
  Rate of time taken wrt to input size. Represented in Big-Oh notation e.g: O(n), O(n2)

1. Always consider worst case scenario.
2. Avoid constants
3. Avoid lower values

Space Complexity:
 Memory space taken by program. Uses Big-Oh notation.

1. Auxilary space ( space required to deal input ) + Input space ( space required to handle inputs)
